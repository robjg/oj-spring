oj-spring 1.2.0
===============

This Oddjob sub project provides support for running Spring from Oddjob.

Requirements
------------

Requires Oddjob 1.2 and Java 1.6

Installation
------------

To use with Oddjob either:

1. Unzip and include lib/*.jar in Oddjobs classpath

or:

2. Unzip into Oddjob's oddballs directory and restart Oddjob.

Documentation
-------------

For more information please see the very limited reference guide and the
javadoc both in the docs directory.
